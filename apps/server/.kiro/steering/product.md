# Product Overview

An Express.js API server that provides AI-powered prompt enhancement services. The primary feature transforms user prompts into more engaging, detailed, and thought-provoking versions using Google's Gemini AI.

## Core Features

- **Prompt Enhancer API**: Takes user prompts and returns enhanced versions with improved clarity, structure, and engagement
- **API Key Validation**: Endpoints accept user-provided Google AI API keys for model access

## Target Use Case

Backend service for applications that need to improve user-generated prompts before sending them to AI models.
